# ROOMS4UMS - SISTEMA V1.0 🚀

Plataforma de foros con una estética retro inspirada en **Windows 95**, construida con tecnologías modernas.

## 🛠️ Tecnologías
- **Frontend:** Angular 19 (Standalone Components, Signals, I18n).
- **Iconografía:** Pixelarticons (vía npm).
- **Estilo:** Vanilla CSS (Custom Properties + Win95 Design System).
- **Backend:** [Aquí puedes poner si es Node.js, Express, etc.]

## 📁 Estructura del Proyecto
```text
rooms4ums/
├── frontend/    # Aplicación de Angular
├── backend/     # API y Lógica de Servidor
└── README.md    # Documentación principal
```

## 🚀 Instalación y Ejecución

### Frontend
1. Entra a la carpeta: `cd frontend`
2. Instala dependencias: `npm install`
3. Corre el servidor: `npm start`
4. Abre: `http://localhost:4200`

### Backend
1. Entra a la carpeta: `cd backend`
2. Instala dependencias: `npm install`
3. Corre el servidor: `npm start` (o el comando que uses)

## 🌐 Funcionalidades Destacadas
- **Internacionalización (i18n):** Soporte dinámico para 5 idiomas (ES, EN, FR, JP, IT).
- **Traducción en Tiempo Real:** Integración con API de MyMemory para traducir posts y comentarios al instante.
- **Retro UI:** Interfaz basada en paneles y ventanas clásicas.
